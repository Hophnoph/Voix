# Voix
Void Linux made Declarative with Simplicity, Minimalism and Ease

Nix + Void = Voix

Voix is a project aimed at creating Nix-style configs for the Void Linux distribution but without all the complexity.

## What is Voix & What Voix Is Not
- Voix makes declarative configurations for Void Linux
- Voix isn't functional (by functional programming)
- Voix doesn't try to be a package manager

## Features
1. Lua configurations
2. You can take snapshots of your config file before applying changes
3. Uses native XBPS package manager
4. Can be easily installed on any Void Linux system
